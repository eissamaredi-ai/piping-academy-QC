ASME
American Society of Mechanical Engineers

استانداردهای طراحی، ساخت و بازرسی تجهیزات مکانیکی و خطوط لوله تحت فشار. در صنعت نفت و گاز بیشتر B31.3 (پایپینگ فرآیندی) و Section IX (تایید صلاحیت جوش) کاربرد دارد.

کدهای مرتبط پرکاربرد
• B31.3 — Process Piping
• Section V — NDE
• Section IX — Welding
• B16.5 / B16.9
حوزه‌ی کاربرد اصلی
پایپینگ، جوشکاری، فلنج و اتصالات
API
American Petroleum Institute

استانداردهای تخصصی صنعت نفت و گاز شامل طراحی مخازن، بازرسی تجهیزات، و الزامات بازرسی دوره‌ای در طول عمر سرویس.

کدهای مرتبط پرکاربرد
• API 510 — Pressure Vessels
• API 570 — Piping Inspection
• API 653 — Tank Inspection
حوزه‌ی کاربرد اصلی
بازرسی مخازن، خطوط لوله سرویس‌دار
ASTM
American Society for Testing and Materials

استاندارد مرجع برای مشخصات متریال — ترکیب شیمیایی، خواص مکانیکی، و روش‌های تست متریال‌های فلزی و غیرفلزی.

کدهای مرتبط پرکاربرد
• A106 — Seamless CS Pipe
• A312 — SS Pipe
• A234 — Fittings
حوزه‌ی کاربرد اصلی
مشخصات متریال لوله، فیتینگ، فلنج
AWS
American Welding Society

استانداردهای جوشکاری شامل طبقه‌بندی الکترود، روش‌های تست، و الزامات صلاحیت جوشکار در صنایع غیرنفتی و عمومی.

کدهای مرتبط پرکاربرد
• D1.1 — Structural Welding
• A5.1 — Electrode Class
حوزه‌ی کاربرد اصلی
جوشکاری سازه‌ای و طبقه‌بندی الکترود
IPS
Iranian Petroleum Standards

استانداردهای داخلی صنعت نفت ایران که اغلب بر اساس استانداردهای بین‌المللی تدوین شده و الزامات بومی پروژه‌های داخلی را پوشش می‌دهند.

کدهای مرتبط پرکاربرد
• IPS-C-PI — Construction
• IPS-E-PI — Engineering
حوزه‌ی کاربرد اصلی
الزامات پروژه‌های داخلی نفت ایران
MSS SP
Manufacturers Standardization Society

استانداردهای ابعادی و عملکردی برای شیرآلات و فیتینگ‌های صنعتی، تکمیل‌کننده‌ی استانداردهای ASME در حوزه‌های جزئی‌تر.

کدهای مرتبط پرکاربرد
• SP-44 — Steel Flanges
• SP-97 — Forged Fittings
حوزه‌ی کاربرد اصلی
ابعاد و مارک‌گذاری شیرآلات
این خلاصه برای آشنایی اولیه است؛ برای الزامات اجرایی دقیق پروژه، همیشه به متن رسمی و آخرین نسخه‌ی استاندارد مراجعه شود.

📘 چرا فقط خلاصه، نه متن کامل استاندارد؟
استانداردهای ASME، API، ASTM و AWS دارای حق نشر هستند و انتشار متن کامل آن‌ها بدون مجوز رسمی غیرقانونی است. این بخش صرفاً برای آشنایی مفهومی طراحی شده — نسخه‌ی رسمی و خرید مجاز هر استاندارد از سایت ناشر مربوطه قابل تهیه است.
