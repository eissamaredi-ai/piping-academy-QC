آزمایش‌های جوش

تست‌های مخرب برای تأیید PQR و تست‌های غیرمخرب برای بازرسی فیلد

آزمایش‌های جوش به دو دسته‌ی اصلی تقسیم می‌شوند: تست‌های مخرب که نمونه را از بین می‌برند و معمولاً فقط برای تأیید PQR استفاده می‌شوند، و تست‌های غیرمخرب که بدون آسیب به قطعه‌ی واقعی پروژه، کیفیت جوش را ارزیابی می‌کنند.

تعریف تست‌های مخرب فقط روی Test Coupon (نمونه‌ی جوش آزمایشی، نه خط واقعی پروژه) برای صدور PQR انجام می‌شوند — هرگز روی جوش واقعی نصب‌شده در فیلد اجرا نمی‌شوند.

تست‌های مخرب

استحکام کششی
Tensile Test
کشش نمونه تا پارگی برای تعیین استحکام کششی نهایی
انعطاف‌پذیری
Bend Test
خم کردن نمونه برای کشف ترک یا عیوب سطحی پنهان در ریشه/رویه
چقرمگی
Impact Test
ضربه به نمونه شیاردار (Charpy) برای سنجش چقرمگی در دمای پایین
سختی
Hardness Test
اندازه‌گیری سختی در نواحی مختلف جوش (Weld, HAZ, Base Metal)
ماکرو ساختار
Etch Test
اچ کردن سطح مقطع برای نمایان‌سازی مرز ذوب و عیوب ماکروسکوپی

عیوب سطحی

ترک سطحی
MT

عیوب نزدیک‌سطح
UT

عیوب داخلی عمیق

ثبت دائمی عیوب داخلی
هشدار انتخاب نوع NDT صرفاً بر اساس دسترسی به دستگاه نباید باشد — مثلاً MT روی استیل ضدزنگ آستنیتی (غیرفرومغناطیس) اصلاً کاربردی ندارد و باید PT یا UT جایگزین شود.
نکته درصد پوشش NDE (مثلاً ۱۰٪ یا ۱۰۰٪ RT) همیشه باید از ITP پروژه استخراج شود، نه یک قاعده‌ی ثابت برای همه‌ی خطوط — این درصد معمولاً بر اساس Service Class، سایز خط و کلاس فشار متغیر است.
