مرور کلی — دو نوع آزمایش با هدف‌های متفاوت

آزمایش‌های جوش به دو دسته اصلی تقسیم می‌شوند: آزمایش‌های مخرب (DT) که روی نمونه‌های آزمایشی جوش (Test Coupon) انجام می‌شوند و هدفشان تأیید WPS در مرحله PQR است — و آزمایش‌های غیرمخرب (NDT) که روی جوش‌های واقعی پروژه در میدان انجام شده و هدفشان یافتن عیوب در اتصالات نهایی است. این دو نقش مکمل هم دارند اما جایگزین یکدیگر نیستند.

📘 قانون بنیادی — ASME B31.3 Para 341.1 «هر جوش باید با بازرسی چشمی (VT) و هر روش NDT مشخص‌شده در کد یا مشخصات پروژه بازرسی شود.» حجم NDT مورد نیاز بر اساس دسته سیال (Fluid Category) طبق B31.3 Table 341.3.2 و درصد آزمایش (Examination Percentage) طبق Para 341.4 تعیین می‌شود. هیچ NDT‌ای نمی‌تواند PQR را جایگزین کند و هیچ PQR‌ای NDT میدانی را.

هدف: تأیید استحکام کششی جوش ≥ حداقل استحکام فلز پایه
ASME Sec. IX QW-150 QW-451 (تعداد) ASTM E8 (روش)
📐 ابعاد نمونه

  • عرض بخش باریک25mm (±0.2mm)
  • ضخامتبرابر t — بدون ماشین‌کاری

  • نمونه در دمای محیط کشیده می‌شود
  • نیروی کشش تا شکست افزایش می‌یابد
  • محل شکست ثبت می‌شود: Weld / HAZ / Base Metal
  • UTS محاسبه: F / A (نیرو / مساحت باریک)
  • شکست در جوش قابل قبول است اگر UTS کافی باشد
📊 معیار پذیرش — QW-153
  • CS — A106 Gr.B≥ 415 MPa (60 ksi)
  • SS — 304L≥ 485 MPa (70 ksi)
  • Duplex 2205≥ 620 MPa (90 ksi)
  • P91 (Grade 91)≥ 585 MPa (85 ksi)
  • محل شکستهر جا — اگر UTS ≥ Min
  • مرجع Min UTSASME Sec. II Part D
✅ معیار نهایی قبول / رد — QW-153
UTS ≥ Min Specified → PASS UTS < Min Specified → FAIL — PQR باطل شکست در Base Metal با UTS کمتر: تنها اگر ≥ 95% UTS min پایه QW-153 + ASME Sec. II Part D Table U
