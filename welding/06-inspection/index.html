📘 تعریف بازرسی فنی جوش — API RP 577 Section 3

طبق API RP 577، بازرسی جوش فرآیندی است که در آن یک متخصص دارای صلاحیت، تمام جنبه‌های جوشکاری شامل مدارک فنی، صلاحیت‌ها، پارامترهای اجرا و کیفیت نهایی را در مقابل الزامات مشخص‌شده ارزیابی می‌کند. بازرس جوش باید از قبل از شروع جوشکاری تا تحویل نهایی پروژه حضور و کنترل داشته باشد — نه فقط در زمان NDT.

نقش‌ها و مسئولیت‌های بازرس جوش

ITP — طرح بازرسی و آزمون (Inspection & Test Plan)

ITP سند اصلی کنترل کیفیت است که تمام فعالیت‌های بازرسی، نقاط توقف (Hold Point)، نقاط گواه (Witness Point) و مدارک لازم را برای هر مرحله پروژه مشخص می‌کند. ITP قراردادی بین Contractor، Client و Third Party Inspector است و باید قبل از شروع کار تأیید همه طرفین را داشته باشد.

🎯 سه نوع نقطه کنترل در ITP

Hold Point (HP): کار نمی‌تواند ادامه یابد مگر با حضور فیزیکی بازرس و صدور تأیید کتبی. مثال: Fit-up قبل از Root Pass، VT نهایی، PWHT، Pressure Test.
Witness Point (WP): بازرس باید اطلاع داشته باشد. کار می‌تواند با اعلام قبلی ادامه یابد. مثال: کنترل Preheat، NDT غیر از اتصالات اصلی.
Review Point (RP): بازرس مدارک را بررسی می‌کند — حضور فیزیکی الزامی نیست. مثال: Mill Certificate، گواهی الکترود.

① Pre-Construction — مرور مدارک

بررسی و تأیید WPS، PQR، WPQ تمام جوشکاران، Mill Certificate متریال، گواهی الکترود و الزامات خاص پروژه (PWHT، Hardness، NDT%). همه اینها باید قبل از شروع هر جوشکاری تأیید شده باشند.

Weld Map — نقشه شناسایی جوش‌ها

Weld Map سندی است که هر اتصال جوش را روی نقشه ایزومتریک یا P&ID شناسه‌گذاری می‌کند. بدون Weld Map، ردیابی کیفیت هر اتصال غیرممکن است. طبق B31.3 Para 328.2.3، هر جوش باید قابل ردیابی به جوشکار و مدارک کیفی آن باشد.

📐 نمونه Weld Map — سیستم شناسه‌گذاری استاندارد

مرجع: B31.3 Para 328.2.3 — هر اتصال جوش باید قابل ردیابی به جوشکار، WPS، PQR و نتایج NDT باشد. Weld Map باید به صورت Live در طول پروژه به‌روز نگه داشته شود — نه فقط در پایان پروژه.

NCR — گزارش عدم انطباق (Non-Conformance Report)

NCR سند رسمی است که صادر می‌شود هرگاه یک محصول، فرآیند یا سند از الزامات مشخص‌شده انحراف داشته باشد. NCR اعلام جنگ نیست — ابزار سیستماتیک بهبود است. صدور به‌موقع NCR از تجمع مشکلات جلوگیری می‌کند.

🚨 موارد صدور فوری NCR — API RP 577 Sec. 10

هر ترک در جوش (هر اندازه) | Undercut از حد مجاز بیشتر | Incomplete Penetration در Severe Cyclic | هر LOF در VT | جوشکاری خارج از محدوده WPQ | جوشکاری بدون WPS معتبر | PWHT انجام نشده در P-No.4 و 5A | Hardness از حد NACE بیشتر | هر عیب که از Table 341.3.2 فراتر رود.

اصول بازرسی چشمی (VT) — مستقل‌ترین ابزار بازرس

VT اولین و اساسی‌ترین روش بازرسی است که باید قبل از هر NDT دیگری انجام شود. طبق ASME Sec. V Art. 9 T-952، VT دارای الزامات دقیق محیطی و فیزیکی است که اگر رعایت نشود، نتیجه آن قابل قبول نیست.

  • ITP نهایی — تمام HP بسته

  • Weld Map نهایی — 100% اتصالات
  • NDT Reports — همه تأیید شده
  • PWHT Charts — تمام متریال‌های الزامی
⚠️ NCR و عدم انطباق
  • تمام NCR Status: Closed
  • Repair Log کامل — NDT تعمیر تأیید شده
  • Punch List نهایی تهیه شده
  • هیچ Open Technical Query باقی نمانده
  • هیچ Conditional Approval باز نیست
  • Client Witness Point‌ها همه امضا شده

  • Pressure Test Package تأیید‌شده
  • فشارسنج کالیبره — گواهی معتبر
  • Relief Valve تنظیم‌شده روی سیستم
  • Blind و Isolation کامل
  • Test Log تهیه شده
  • بازرس Client یا TPI حضور دارد

چک‌لیست نهایی بازرس جوش — خلاصه کل دوره

این چک‌لیست خلاصه تمام مهارت‌های عملی است که یک بازرس جوش حرفه‌ای در پروژه‌های ASME B31.3 باید داشته باشد.

چک‌لیست جامع بازرس جوش — از WPS تا تحویل

مرجع: ASME B31.3 Para 328–346 | API RP 577 Sec. 3–12 | AWS B1.11

1
WPS/PQR/WPQ بررسی: قبل از هر جوش — همه تأیید شده و در محدوده باشندB31.3 Para 328 + ASME Sec. IX QW-300
2
Fit-up Hold Point: Root Gap، Hi-Lo، Bevel، تمیزی — قبل از اولین قوسPara 328.4.3 — Hi-Lo ≤ 1.6mm
3
Preheat Contact Pyrometer: ≥75mm از لبه — هر دو طرف — قبل از Root PassPara 330.1.2 + Table 330.1.1
4
Stamp جوشکار: شناسه روی هر اتصال — ردیابی به WPQ الزامی استPara 328.2.3 + Weld Map
5
Interpass Temperature: قبل از هر پاس — Contact Pyrometer — ثبت در Welding LogASME Sec. IX QW-406.3
6
VT بین پاس: Slag کاملاً برداشته — هر Crack یا Porosity → Stop WorkPara 341.4.1 + API RP 577 Sec. 8
7
Back Purge SS: O₂ ≤ 0.1% با Analyzer قبل از Root — کنترل مداومAPI RP 577 Sec. 7.2
8
VT نهایی با ≥500 Lux: Pit Gauge برای Undercut — Weld Gauge برای ReinforcementSec. V Art. 9 T-952 + Table 341.3.2
9
Written NDT Procedure: قبل از اجرا تأیید شده — Operator Level II — تجهیز کالیبرهSec. V Art. 1 T-150
10
PWHT با Thermocouple مستقیم: Chart Recorder کالیبره — نرخ ≤ 222°C/hrPara 331.1.3 + Table 331.1.1
11
Hardness Survey (Sour): HAZ ≤ 248 HV10 پس از PWHT — Portable TesterNACE MR0175 / ISO 15156-2 Table B.1
12
NCR بسته‌شده: تمام NCR قبل از Pressure Test — Weld Map به‌روزAPI RP 577 Sec. 10 + Para 346
13
Data Book کامل: ITP + WPS/PQR + Weld Map + NDT + PWHT + NCR → قبل از MCB31.3 Para 346.2
14
Pressure Test Package: P_test = 1.5 × MAWP × (St/Sd) — فشارسنج کالیبرهB31.3 Para 345.4.2
🚨 پنج خط قرمز بازرس جوش — هرگز فراموش نکنید

(1) هیچ جوشکاری خارج از محدوده WPQ — حتی اگر فقط یک پاس باشد.
(2) هیچ تعمیر جوش بدون NCR صادره و Repair WPS — ثبت همه چیز.
(3) PWHT برای P-No.4 و 5A بدون Chart Recorder و Thermocouple مستقیم — کافی نیست.
(4) RT برای تأیید LOF — اشتباه فنی است؛ UT TOFD استفاده شود.
(5) امضای Data Book ناقص — بازرس مسئول است؛ فقط زمانی که همه موارد کامل باشند.
