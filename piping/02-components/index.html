اجزای پایپینگ

شناخت کامل اجزای سیستم لوله‌کشی صنعتی — از لوله تا فلنج، شیر و اتصالات

یک سیستم پایپینگ صنعتی از مجموعه‌ای از اجزای استاندارد تشکیل شده که هر کدام وظیفه مشخصی دارند. شناخت دقیق این اجزا، استانداردهای حاکم بر آن‌ها و معیارهای انتخاب صحیح، پایه‌ی هر بازرسی و هر تصمیم فنی در پروژه است.

نکته بحرانی — تطبیق Rating تمام اجزای یک سیستم پایپینگ باید از نظر Pressure Rating، Temperature Rating و Material Class با یکدیگر سازگار باشند. طبق ASME B31.3 Para. 305، اتصال اجزا با Rating‌های ناسازگار یک نقص طراحی بحرانی است.

۱. لوله (Pipe)

P
Pipe عنصر اصلی انتقال سیال

لوله قطعه اصلی هر سیستم پایپینگ است. اندازه آن با دو پارامتر تعریف می‌شود: NPS (Nominal Pipe Size) برای قطر اسمی و Schedule برای ضخامت دیواره. این دو پارامتر با هم Wall Thickness و در نتیجه Pressure Rating خط را تعیین می‌کنند.

تعریف اندازهNPS + Schedule → Wall Thickness (ASME B36.10M)
محدوده NPSاز NPS ½" تا NPS 80"
Schedule‌های رایجSCH 40، SCH 80، SCH 160، XXS
نوع ساختSeamless (بدون درز) یا ERW/EFW (درزدار)
نکته — Seamless vs Welded طبق ASME B31.3 Table A-1B، لوله‌های درزدار (ERW/EFW) دارای Quality Factor (Ej) کمتر از ۱.۰ هستند. برای سرویس‌های بحرانی یا فشار بالا، لوله Seamless الزامی است.

۲. اتصالات (Fittings)

F
Fittings تغییر مسیر، انشعاب و تغییر اندازه

اتصالات قطعاتی هستند که مسیر، اندازه یا انشعاب خط لوله را تغییر می‌دهند. تمام Fittings باید با لوله از نظر Material، Rating و End Connection سازگار باشند. استاندارد ابعادی اصلی ASME B16.9 برای BW Fittings و ASME B16.11 برای SW/Threaded Fittings است.

۳. فلنج‌ها (Flanges)

FL
Flanges اتصال جداشدنی — برای تعمیر و نگهداری

فلنج اتصال Demountable سیستم پایپینگ است. انتخاب نوع فلنج بر اساس فشار، دما، سیال و نوع جوشکاری مجاز است. استاندارد اصلی ASME B16.5 برای NPS ½" تا NPS 24" و ASME B16.47 برای سایزهای بزرگتر است.

Weld Neck Flange (WN)
فلنج گردن جوشی
بهترین انتخاب برای High Pressure، High Temp و Cyclic. جوش BW با لوله — توزیع تنش عالی.
ASME B16.5 — پیشنهادی Class 600 به بالا
Slip-On Flange (SO)
فلنج اسلیپ‌اون
دو جوش Fillet داخل و خارج. ارزان‌تر از WN — Rating پایین‌تر. مناسب سرویس‌های کم‌فشار.
ASME B16.5 — محدود در سرویس‌های بحرانی
Socket Weld Flange (SW)
فلنج سوکت‌ولد
برای NPS ≤ 2". Gap 1.6mm قبل از جوش الزامی — ASME B31.3 Para. 328.5.2.
ASME B16.5 — NPS ≤ 2" فقط
Blind Flange (BL)
فلنج کور
مسدودکردن انتهای خط یا نازل تجهیز. در Hydrotest و Line Blinding استفاده می‌شود.
ASME B16.5
Lap Joint Flange
فلنج لپ‌جوینت
با Stub End استفاده می‌شود. فلنج Loose می‌چرخد — تنظیم سوراخ‌های Bolt آسان. برای SS در کنار CS.
ASME B16.5 + ASME B16.9 (Stub End)

۴. شیرآلات (Valves)

V
Valves کنترل جریان، ایزولاسیون و تنظیم فشار

شیرآلات برای کنترل، ایزولاسیون یا تنظیم جریان سیال در سیستم پایپینگ استفاده می‌شوند. انتخاب نوع شیر بر اساس عملکرد مورد نیاز (On/Off یا Control)، سیال، فشار، دما و فرکانس عملکرد است.

Gate Valve
شیر دروازه‌ای
فقط برای On/Off — نه Throttling. افت فشار بسیار پایین در حالت کاملاً باز. برای ایزولاسیون خطوط.
API 600 / API 603 / ASME B16.34
Globe Valve
شیر کروی
برای Throttling و تنظیم جریان. افت فشار بالا. مناسب سرویس‌های Steam و بخار.
ASME B16.34
Ball Valve
شیر توپی
On/Off سریع با چرخش ۹۰ درجه. Bore کامل (Full Bore) یا کاهش‌یافته (Reduced Bore). رایج در پایپینگ فرآیندی.
API 6D / ASME B16.34
Butterfly Valve
شیر پروانه‌ای
سبک و ارزان — برای قطرهای بزرگ. Triple Offset برای سرویس‌های بحرانی و High Temp.
API 609 / ASME B16.34
Check Valve
شیر یک‌طرفه
جلوگیری از برگشت جریان. Swing، Tilting Disc یا Piston نوع — انتخاب بر اساس سرویس.
API 594 / ASME B16.34
Safety / Relief Valve
شیر اطمینان
حفاظت در برابر فشار بیش از حد. Set Pressure طبق طراحی. تست و تنظیم دوره‌ای الزامی.
ASME Section VIII / API 520 / API 526
