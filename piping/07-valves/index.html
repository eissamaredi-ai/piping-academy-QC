PIPING VALVES

شیرآلات پایپینگ

انواع شیرآلات صنعتی، استانداردهای API و ASME، نحوه انتخاب، نصب، تست و بازرسی شیرآلات در پروژه‌های نفت، گاز و پتروشیمی

شیرآلات (Valves) قلب هر سیستم پایپینگ هستند. وظیفه اصلی آن‌ها کنترل جریان سیال — از نظر مقدار، جهت، فشار و دما — است. انتخاب نادرست شیر می‌تواند منجر به نشتی، کاویتاسیون، Water Hammer و حتی حوادث فاجعه‌بار شود. در این فصل با انواع شیرآلات، استانداردها و اصول بازرسی آن‌ها آشنا می‌شوید.

انواع شیرآلات صنعتی

هشت نوع اصلی شیرآلات مورد استفاده در پایپینگ صنعتی

📘 مقدمه شیرآلات بر اساس نوع عملکرد به سه دسته تقسیم می‌شوند:
• On/Off (Isolation) — قطع و وصل کامل جریان
• Throttling (Control) — تنظیم مقدار جریان
• Non-Return (Check) — جلوگیری از برگشت جریان
GATE
شیر دروازه‌ای
Gate Valve
برای قطع و وصل کامل جریان (On/Off). برای Throttling مناسب نیست چون باعث ارتعاش و خوردگی Disc می‌شود.
کاربرد: خطوط اصلی، Isolation، سرویس‌های با فشار بالا
GLOBE
شیر کروی
Globe Valve
مناسب برای تنظیم جریان (Throttling) و قطع دقیق. افت فشار بالاتر نسبت به Gate Valve دارد.
کاربرد: کنترل جریان، سیستم‌های بخار، Cooling Water
BALL
شیر توپی
Ball Valve
عملکرد سریع با چرخش ۹۰ درجه. Leak-proof عالی. مناسب برای گاز و سیالات خورنده.
کاربرد: خطوط گاز، Oil & Gas، سرویس‌های On/Off سریع
BUTTERFLY
شیر پروانه‌ای
Butterfly Valve
سبک، ارزان و مناسب برای قطرهای بزرگ. هم برای On/Off و هم Throttling استفاده می‌شود.
کاربرد: آب، هوا، خطوط با قطر بالا (DN300+)
CHECK
شیر یک‌طرفه
Check Valve (NRV)
جلوگیری از برگشت جریان (Backflow). نصب بدون نیاز به اپراتور. انواع: Swing, Lift, Wafer, Dual Plate.
کاربرد: خروجی پمپ‌ها، کمپرسورها، خطوط با خطر Backflow
PLUG
شیر پلاگی
Plug Valve
ساده، سریع و مناسب برای سیالات غلیظ و Slurry. عملکرد مشابه Ball Valve ولی با Plug مخروطی.
کاربرد: Oil & Gas، Slurry، خطوط با ذرات معلق
NEEDLE
شیر سوزنی
Needle Valve
برای تنظیم دقیق جریان در قطرهای کوچک. معمولاً در Instrument Lines و Sampling استفاده می‌شود.
کاربرد: Instrumentation، Gauge Lines، Sampling Points
PSV/PRV
شیر اطمینان
Safety / Relief Valve
محافظت از تجهیزات در برابر فشار بیش از حد (Overpressure). تنظیم‌شده روی Set Pressure مشخص.
کاربرد: مخازن، بویلرها، خطوط با فشار بحرانی

استانداردهای شیرآلات

مهم‌ترین استانداردهای بین‌المللی طراحی و تست شیرآلات

API 600
Gate Valves - Flanged & Butt-Weld
شیرهای دروازه‌ای فلنجی و جوشی برای صنایع نفت و گاز. پوشش بدنه، متریال و ابعاد.
API 602
Compact Gate Valves (≤ NPS 2)
شیرهای دروازه‌ای کوچک (Compact) برای سایزهای ۲ اینچ و کمتر. Forge یا Cast.
API 607
Fire Test for Soft-Seated Valves
تست آتش برای شیرهای با Seat نرم. اطمینان از عملکرد در شرایط حریق.
API 608
Metal Ball Valves
شیرهای توپی فلزی فلنجی، رزوه‌ای و جوشی. الزامات طراحی و تست.
API 609
Butterfly Valves
شیرهای پروانه‌ای Double Offset و Triple Offset برای فشار و دمای بالا.
ASME B16.34
Valves - Flanged, Threaded, Welding
استاندارد مادر برای Pressure-Temperature Rating و Wall Thickness شیرآلات.
API 598
Valve Inspection & Testing
الزامات تست فشار، Seat Leakage و بازرسی نهایی تمام شیرآلات.
API 526/527
Pressure Relief Valves
طراحی، انتخاب و تست شیرهای اطمینان (PSV/PRV) برای Overpressure Protection.

تست و بازرسی شیرآلات

الزامات تست طبق API 598 و ASME B16.34

🛠 چک‌لیست بازرسی شیرآلات (Valve Inspection)
بررسی Body & Bonnet Material Certificate (MTC)
کنترل ابعاد Face-to-Face طبق API 600 / ASME B16.10
بازرسی جوش Body-Bonnet (VT + PT/MT)
تست Shell Test (1.5 × Design Pressure)
تست Seat Leakage (API 598 Rate A/B/C/D)
بررسی عملکرد دستی (Handwheel Operation)
کنترل Painting & Coating Thickness
تأیید Nameplate (Size, Class, Material, Std)
💡 نکته حرفه‌ای در پروژه‌های نفت و گاز، شیرآلات Critical Service (مثل Hydrocarbon, H₂S, High Pressure) باید 100% Shell Test و Seat Test شوند. برای شیرهای Non-Critical می‌توان از Sampling Test (حداقل 10%) استفاده کرد.

مشکلات رایج شیرآلات

عیوبی که بازرس باید در حین بازرسی شناسایی کند

نشتی داخلی
Internal Leakage
خرابی Seat یا Disc. معمولاً به دلیل ذرات جامد، خوردگی یا نصب نادرست.
نشتی خارجی
External Leakage
خرابی Packing یا Gasket Body-Bonnet. نیاز به Re-torque یا تعویض Packing.
کاویتاسیون
Cavitation
افت فشار شدید در Throttling. باعث خوردگی شدید Trim و لرزش شیر می‌شود.
Water Hammer
ضربه قوچ
بسته شدن سریع شیر. باعث فشار لحظه‌ای بالا و آسیب به لوله و شیر می‌شود.
جمع‌بندی فصل انتخاب صحیح شیرآلات بر اساس نوع سیال، فشار، دما، سایز خط و نوع عملکرد انجام می‌شود. هر شیر استانداردهای خاص خود را دارد و بازرسی آن باید طبق API 598 انجام شود. شناخت Trim، Body Material و Pressure Class برای هر بازرس فنی الزامی است.
